Excel actual vs Airtable

Por qué el Excel que tienen ya no es suficiente
Excel actual — Limitaciones
  • Solo pueden escribir y ver datos en vivo
  • No pueden generar reportes
  • No pueden imprimir ni exportar en formato útil
  • No hay historial de cambios (¿quién cambió qué?)
  • No se conecta al sitio web
  • No envía notificaciones automáticas
  • Sin control de acceso (todos ven todo)
  • Sin vistas por departamento
  • Difícil de mantener cuando crece el negocio
Airtable — Lo que ganan
  • Agregan, editan y eliminan datos fácilmente
  • Reportes automáticos por dealer, modelo, fecha
  • Exportar a PDF, CSV, Excel en un clic
  • Historial completo de cambios con fecha y usuario
  • Se conecta al sitio web — stock se actualiza solo
  • Notificaciones por correo cuando stock baja
  • Control de acceso por rol (admin, ventas, etc.)
  • Vistas personalizadas por departamento
  • Escala con el negocio sin costo adicional inicial

Cómo empezar

4 pasos — el equipo puede tener esto funcionando en 1 día
1

Crear cuenta gratis

Ir a airtable.com — registrarse con el correo de SG Trailers. Plan Free es suficiente para empezar.

2

Migrar el Excel

Airtable permite importar directamente desde Excel/CSV. Los datos actuales de refacciones se importan en minutos.

3

Crear las 4 tablas

Dealers, Inventario, Refacciones y Leads. El desarrollador configura los campos y las relaciones entre tablas.

4

Conectar al sitio

El desarrollador conecta la API de Airtable al sitio. A partir de ahí el equipo actualiza datos y el sitio se actualiza solo.

Costo: Plan gratuito de Airtable es suficiente para empezar (hasta 1,000 registros por tabla). Si el catálogo de refacciones crece mucho, el plan Team cuesta $20/mes. La conexión con el sitio web es trabajo del desarrollador — se hace una sola vez.
